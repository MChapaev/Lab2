Задание 1.
1. Описать 5 методов множеств.
Ответ:
.add(x) - добавляет элемент x во множество
.remove(x) - удаляет элемент x из множества. Если элемент не найден, вызывает ошибку
.discard(x) - удаляет x, но не вызывает ошибку
set1.union(set2) - возвращает множество, состоящее из элементов set1 и set2
set1.intersection - возвращает общие элементы множеств set1 и set2

2. Опишите известные Вам способы задать множество.
Ответ:
set1 = {1, 2, 3}
set1 = set([1, 2, 3])
set1 = set()

Задание 2.
1. Описать 5 методов словарей.
Ответ:
.get(key, default=None) — возвращает значение по ключу, если его нет — default
.keys() — возвращает все ключи словаря
.values() — возвращает все значения словаря
.items() — возвращает пары (ключ, значение)
.pop(key, default=None) — удаляет ключ и возвращает его значение, если ключ отсутствует — возвращает default

2. Опишите известные Вам способы задать словарь.
Ответ:
dict1 = {'a': 1, 'b': 2, 'c': 3}
dict1 = dict(a=1, b=2, c=3)
dict1 = dict([('a', 1), ('b', 2), ('c', 3)])
dict1 = {x: x for x in range(1, 6)}
